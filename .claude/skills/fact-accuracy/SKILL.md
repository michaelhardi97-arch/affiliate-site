# Fact Accuracy

Für jede faktische Angabe gilt:
1. Niemals aus dem Gedächtnis raten. Wert gegen die offizielle Quelle prüfen (Anbieter-Website / deren Hilfeseite) per Web-Recherche. Ist keine Verifikation möglich, allgemein formulieren statt eine Zahl zu erfinden.
2. Aktuelles Jahr verwenden (2026). Keine 2024/2025-Angaben für aktuelle Sachverhalte.
3. Datenführende Seiten bekommen "Stand: [Monat] 2026".
4. Variable Werte als "bis zu"/"ab" kennzeichnen, nicht als Fixwert.
5. Keine übertriebenen Verdienstversprechen.
